# Day 1 - Calculator

## What I Learned
- **Data types**:
  - **String** - A sequence of characters used to store text data.
  - **Integer** - A whole number without a decimal point.
  - **Float** - A number with a decimal point.
  - **Boolean (bool)** - A data type that can have only two values: `True` or `False`.

- **Indexing**:
  - In Python, indexing starts at `0`, not `1`.
  - Negative indexes access characters from the end of a string.

- **Large numbers**:
  - Underscores (`_`) can be used to make large numbers easier to read.

- **Mathematical operations**:
  - **Addition** (`+`) - Adds two numbers.
  - **Subtraction** (`-`) - Subtracts one number from another.
  - **Multiplication** (`*`) - Multiplies two numbers.
  - **Division** (`/`) - Divides one number by another.
  - **Exponentiation** (`**`) - Raises a number to a power.
  - **Floor division** (`//`) - Divides and rounds down to the nearest whole number.
  - **Modulo** (`%`) - Returns the remainder after division.

- **Order of operations**:
  - Python follows the standard mathematical order of operations:
    1. Parentheses `()`
    2. Exponentiation `**`
    3. Multiplication `*`, Division `/`, Floor division `//`, Modulo `%`
    4. Addition `+`, Subtraction `-`

- **Functions**:
  - **round()** - Rounds a float to the nearest whole number or to the specified number of decimal places.
  - **type()** - Returns the data type of a value.

- **f-strings**:
  - Used to insert variables and expressions directly into a string.
